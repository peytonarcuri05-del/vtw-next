# VTW — vs the world (Next.js App Router)

Dark/purple streetwear landing page for VTW.

## Run locally

```bash
pnpm i   # or npm i / yarn
pnpm dev # http://localhost:3000
```

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- framer-motion
- lucide-react

## Where to edit
- `app/page.tsx` — page content
- `components/ui/*` — minimal UI primitives
- `styles/globals.css` — base styles

Replace placeholder images with your product/hero shots.
Connect the form and newsletter to your backend or a service (e.g., Resend, Formspree, Mailchimp).
